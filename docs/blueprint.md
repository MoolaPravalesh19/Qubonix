# **App Name**: Qubnix Hub

## Core Features:

- Service Showcase: Display IT services with icons on the Services page, providing clear descriptions for each.
- Internship Program Details: Explain the internship program benefits, technologies used, and duration on the Internship page.
- Project Portfolio Display: Showcase past projects on the Portfolio page to demonstrate the company's capabilities.
- Intern Login/Signup: Allow interns to create accounts and log in to the dashboard.
- Task Management: Enable interns to view assigned tasks, submit work, and check their internship status.
- Authentication: Implement secure login and signup using Firebase Authentication.
- Data Storage: Use Firebase Firestore to store intern data, tasks, and progress updates.

## Style Guidelines:

- Primary color: Use a vibrant orange (#FF8C00), inspired by the Qubnix logo, to convey innovation and energy.
- Background color: A very light, desaturated tint of orange (#FAF9F6) provides a clean, modern backdrop that doesn't distract from the content.
- Accent color: Analogous to the primary, a slightly more saturated orange (#FF7700) is used to highlight key interactive elements and call-to-action buttons.
- Headline font: 'Space Grotesk' for a contemporary, precise feel.
- Body font: 'Inter', sans-serif for a clean, readable, modern style
- Use modern, minimalist icons to represent services and features.
- Maintain a clean and corporate layout with ample whitespace for a professional look.
- Incorporate subtle transitions and animations to enhance user experience and engagement.
